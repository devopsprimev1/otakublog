# OtakuBlog 🌸

Portal de cultura otaku e anime com reviews, notícias e rankings.

## Stack
- **Astro** + **Tailwind CSS** + **TypeScript**
- Dark theme com neon accents (rosa/roxo/cyan)
- Deploy: Vercel

## Pages
- `/` - Home com posts em destaque
- `/reviews` - Reviews completos de animes
- `/news` - Notícias em tempo real
- `/rankings` - Top 10 animes

## Dev
```bash
npm install
npm run dev
npm run build
```
